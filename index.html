Prefix

O prefixo padrão de Nyohi começa com /

Os comandos são marcados com base nas permissões necessárias para sua execução. Comandos marcados como Not for DM não pode ser usado em mensagens diretas. Comandos marcados como Admin requer as permissões de administrador do Discord. Comandos marcados com purple tags requer a respectiva permissão que destaca. Comandos marcados como Everyone não requer permissões.

Certos comandos também precisam ser configurados antes de serem utilizáveis. Comandos marcados como NSFW requer que o canal em que o comando foi usado seja do tipo nsfw.
